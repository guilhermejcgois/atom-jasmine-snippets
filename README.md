# Jasmine snippets for Atom

This is a collection of jasmine statements common in my day by day.

Found some typo? Miss something else cool? Let me now or make a pull request!

# Install

`apm install jasmine-snippets`

# Supported languages

- [ ] JavaScript
- [ ] Python
- [ ] Ruby
- [x] TypeScript
